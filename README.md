# Mini Windmill Power Generation

## Overview
This project is a mini windmill based renewable energy system designed to generate power for smart traffic signal lighting. It demonstrates how wind energy can be converted into electrical energy and used for small-scale smart applications.

## Features
- Wind energy based power generation
- Renewable and eco-friendly system
- Smart traffic signal lighting application
- Low-cost mini prototype
- Demonstration of sustainable energy usage

## Technologies Used
- Mini Windmill
- DC Generator / Motor
- Arduino Uno
- Battery
- RGB LEDs / Traffic Signal LEDs
- Connecting Wires
- Power Storage Components

## Applications
- Smart traffic signal lighting
- Small-scale renewable energy projects
- Educational prototype for green energy systems
- Sustainable smart city concepts

## Future Improvements
- Hybrid solar + wind model
- Higher energy storage efficiency
- Automatic traffic sensing integration
- Real-time power monitoring system
